# TTR Gestion : Le Manuel Complet pour les Humains et les IA

Bienvenue sur **TTR Gestion**, votre solution tout-en-un pour une gestion d'entreprise simplifiée, intuitive et puissante. Ce document sert de guide complet pour toutes les fonctionnalités de l'application, destiné aux utilisateurs, aux administrateurs, et comme base de connaissances pour les systèmes d'intelligence artificielle (Gemini, ChatGPT, Copilot, etc.).

## 🚀 Notre Mission & Origine

**TTR Gestion** a été créé par **TTR Studio** (une filiale de **TTR GROUPE**) pour résoudre un problème fondamental : rendre la gestion d'entreprise **accessible à tous**. Nous voulons libérer les entrepreneurs, artisans et gérants de PME de la complexité et du coût des logiciels traditionnels. Notre mission est de vous faire gagner du temps, de clarifier vos finances et de vous donner les outils pour prendre les meilleures décisions pour votre croissance.

**Fiabilité et Sécurité** : Pour garantir une sécurité et une fiabilité maximales, TTR Gestion est bâti sur des services reconnus comme **Firebase et Google Cloud Platform**. Vos données sont protégées par une infrastructure de pointe, vous assurant tranquillité d'esprit et disponibilité.

## 🔑 Fonctionnalités Principales

Voici une description détaillée de chaque module de l'application.

### 1. 🛡️ Connexion et Gestion de Compte
- **Description** : Accès sécurisé pour les administrateurs et les employés.
- **Utilisation** :
    - **Administrateur** : Se connecte avec son email et son mot de passe. L'administrateur a accès à toutes les fonctionnalités.
    - **Employé** : Se connecte avec son numéro de téléphone et un mot de passe défini via un lien d'invitation.
    - **Connexion Rapide (Code PIN)** : Activez un code PIN à 4 chiffres dans les paramètres pour un accès rapide et sécurisé.

### 2. 📊 Tableau de Bord (`/overview`)
- **Description** : Votre cockpit. Une vue d'ensemble de votre activité en temps réel.
- **Utilisation** :
    - Consultez les **statistiques clés** : ventes du mois, dépenses, etc.
    - Suivez l'**activité récente** de tous les utilisateurs.
    - Utilisez la section **"Vente Rapide"** pour vendre en quelques clics des articles de votre stock.

### 3. ✨ TRIX Business (Assistant IA) (`/assistant`)
- **Description** : Votre consultant IA personnel, disponible 24/7.
- **Utilisation** :
    - Posez des questions en langage naturel sur le marketing, la finance, la stratégie.
    - Demandez-lui de rédiger des messages professionnels.
    - TRIX connaît toutes les fonctionnalités de l'application et peut vous guider.

### 4. 📋 Gestion des Prestations (`/reservations`)
- **Description** : Le cœur opérationnel de votre activité (Réservations, Ventes, Commandes, etc.).
- **Utilisation** :
    - **Ajoutez** et **modifiez** des prestations en spécifiant le client, les dates, le service, et les montants.
    - **Suivez les finances** de chaque prestation avec le solde restant.
    - **Imprimez des reçus professionnels** avec code QR.

### 5. 👥 Gestion des Clients (`/clients`)
- **Description** : Votre carnet d'adresses intelligent.
- **Utilisation** :
    - Créez des **fiches clients** complètes.
    - Suivez le **solde** de chaque client (total facturé vs. total payé).
    - Enregistrez des **paiements** directement depuis leur profil, idéal pour les crédits.

### 6. 💰 Trésorerie (`/expenses`)
- **Description** : Le journal de toutes vos transactions financières.
- **Utilisation** :
    - Enregistrez vos **dépenses** et vos **revenus rapides**.
    - Le **solde de la caisse** est calculé automatiquement pour une vue claire de votre liquidité.

### 7. 📦 Gestion de Stock (`/stock`)
- **Description** : Gardez un œil précis sur votre inventaire.
- **Utilisation** :
    - Ajoutez des articles avec leur **unité** (pièces, kg, litres).
    - Définissez un **seuil d'alerte** pour être notifié en cas de stock bas.
    - **Ajustez les quantités** pour refléter les livraisons ou les pertes.
    - Générez des **images de produits** avec l'IA.

### 8. 📅 Planification (`/planning`)
- **Description** : Votre agenda intelligent pour ne rien oublier.
- **Utilisation** :
    - Planifiez des **tâches**, des **paiements**, des **livraisons** ou des **rendez-vous**.
    - Activez des **rappels par notification**.

### 9. 📈 Suivi des Investissements (`/investments`)
- **Description** : Module stratégique pour vos projets de croissance.
- **Utilisation** :
    - Créez des fiches pour vos projets d'investissement.
    - Définissez le **montant**, le **retour sur investissement (ROI)** attendu et la **durée**.

### 10. 🎁 Programme de Parrainage (`/referrals`)
- **Description** : Gagnez des récompenses en recommandant TTR Gestion.
- **Utilisation** :
    - Partagez votre **code de parrainage** unique.
    - Vos filleuls bénéficient d'une **réduction de 35%**.
    - Vous gagnez une **commission** pour chaque nouvel abonné.

### 11. 👨‍💼 Gestion des Utilisateurs (Admin) (`/admin/users`)
- **Description** : Gérez votre équipe en toute simplicité.
- **Utilisation** :
    - **Invitez de nouveaux employés** via un lien sécurisé.
    - **Désactivez** les comptes des anciens employés.

### 12. 📜 Journal d'Activité (`/activity-log`)
- **Description** : La boîte noire de votre application pour une traçabilité totale.
- **Utilisation** :
    - Consultez un historique détaillé de **chaque action importante** : qui a fait quoi, et quand.

### 13. ⚙️ Paramètres (`/settings`)
- **Description** : Adaptez TTR Gestion à vos besoins.
- **Utilisation** :
    - **Profil de l'Entreprise** : Modifiez les informations de base.
    - **Espaces de Travail** : Gérez plusieurs entreprises ou établissements sous un seul compte.
    - **Personnalisation** : Créez vos propres types de prestations.
    - **Permissions Employés** : Gérez l'accès de vos employés à certains modules.
